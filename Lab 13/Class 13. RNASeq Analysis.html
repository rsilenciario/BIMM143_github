control.counts <- counts[,control.inds]

Calculate the mean value for each gene in these “control” columns

control.mean <- rowMeans(control.counts)

Hold on, we are missing stats. Is it actually significant? We need to calculate the p-value to make sure of this.

#DESeq2 analysis

Let’s do this properly and consider the stats - are the differences in the means significant?

We will use DESeq2 to do this:

library(DESeq2)

The first function we will use for this package sets up the input in the particular format that DESeq2 wants:

dds <- DESeqDataSetFromMatrix(countData = counts, colData= metadata, design = ~dex)
